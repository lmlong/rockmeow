# Heartbeat 任务

心跳服务每30分钟读取此文件并执行其中的指令。

## 日程检查

检查未来2小时的日历事件，如果有即将开始的会议请通过飞书提醒我。

## 系统监控

检查服务器状态：
- 磁盘空间（df -h）
- 内存使用（free -h）
- 如果异常，发送飞书告警

## 每日记忆提炼（凌晨 2:00 执行）

当前时间如果在 2:00-2:30 之间，执行记忆归档提炼：

1. 使用 memory 工具执行：`{"action": "refine", "archiveOld": true, "recentDays": 3}`
2. 这会合并超过 3 天滑动窗口的旧记忆，去除重复条目
3. 提炼完成后，发送简报到飞书（包含合并了多少条、删除了多少重复）

注意：只在每天第一次检测到此时间段时执行，避免重复执行。

当前时间如果在 9:00-9:30 之间，抓取以下网站的 Top 3 热门文章，汇总后发送到飞书：

1. 机器之心 - https://www.jiqizhixin.com/
2. 量子位 - https://www.qbitai.com/
3. 新智元 - https://www.163.com/dy/media/1623.html

格式要求：
- 每个网站列出 3 篇最热门/最新的文章
- 包含标题、链接和简要摘要（如有）
- 按「网站名」分组展示
